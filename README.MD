# Example Krunker Client
## About
#### This is a start off [Krunker](https://krunker.io) client using [NodeJS](https://nodejs.org/en/) and [Electron](https://www.electronjs.org/).
## How to setup
1. Install NodeJS
2. Paste `npm i electron --save` in terminal in this folder
3. Modify the code
